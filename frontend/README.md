# Frontend

React 19 + TypeScript + Vite frontend for the Probabilistic AI Lab.

## Development

```bash
npm install
npm run dev
```

The dev server runs on `http://localhost:5173` and proxies `/api` requests to the FastAPI backend on `http://localhost:8000`.

## Available scripts

- `npm run dev` starts the Vite dev server
- `npm run build` creates a production-style bundle in `dist/`
- `npm run preview` serves the built bundle locally
- `npm run lint` runs ESLint
